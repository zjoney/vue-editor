# git 提交规范

## commit

- 💥 feat(模块): 添加了个很棒的功能
- 🐛 fix(模块): 修复了一些 bug
- 📝 docs(模块): 更新了一下文档
- 🌷 style(模块): 修改了一下样式
- ♻️ refactor : 某个已有功能重构
- ⚡️ perf : 性能优化
- ✅ test : 增加测试
- 🔧 build : 改变了build工具 如 grunt换成了 npm
- 🏰 chore(模块): 对脚手架做了些更改
- 🌐 locale(模块): 为国际化做了微小的贡献
